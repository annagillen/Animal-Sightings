Description of Overall Project:

This project will be a web application that collects real time animal sighting data from hikers and backpackers of national parks. Hikers can create an account and keep track of their sightings as well as see data from other users. Users will be able to search based on parks, trails, and animals sighted to inform their activities. Data collected will be presented in a graphical informational way.

Project Relevance:

This proposal is linked to the overall goals of this class because it will require project management, version control, building, testing, and issue tracking. It will also involve front end development, back end development, database management, and debugging.
